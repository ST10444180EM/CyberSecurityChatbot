# Cybersecurity Awareness Chatbot (Part 1)


# About This Project 

This project is a simple console-based chatbot developed in C# as part of my PROG assignment. The chatbot focuses on raising awareness about cybersecurity by interacting with users in a friendly and informative way. It provides basic guidance on topics such as password safety, phishing, and safe browsing.
The goal of this part of the project was to build a functional chatbot with a clean structure, user interaction, and enhanced console features like audio and visual elements.



# Features Implemented

# Voice Greeting
- A recorded welcome message (`welcome.wav`) plays when the application starts.
- This adds a personal and interactive touch to the chatbot.


# ASCII Art Display
- A cybersecurity themed ASCII design is displayed at launch.
- Acts as a visual header to introduce the chatbot.


# User Interaction
- The chatbot asks the user for their name.
- Responses are personalized using the user’s name to create a more engaging experience.


# Basic Chatbot Responses
The chatbot responds to common questions such as:

- “How are you?”
- “What is your purpose?”
- “What can I ask you about?”

It also provides cybersecurity tips on:
- Password safety
- Phishing awareness
- Safe browsing
  

# Input Validation

- Handles empty inputs and unknown queries.
- Provides a default response when it doesn’t understand the user.

# Enhanced Console UI

- Uses colors, spacing, and formatting for better readability.
- Includes a typing effect to simulate a real conversation.

# Code Structure

The project is structured using multiple classes for clarity and maintainability:

* "Program.cs" – Entry point of the application
* "Chatbot.cs" – Handles chatbot logic and responses
* "UIHelper.cs" – Manages UI elements like typing effect and display
* "AudioHelper.cs" – Handles audio playback


# The Technologies Used

- C#
- .NET Console Application
- System.Media ,for audio playback


# GitHub

- The project is managed using GitHub for version control.
- Multiple commits were made to track progress and improvements.
- GitHub Actions (CI) is set up to ensure the project builds successfully with each push.



# How to Run My Project

1. Open the project in Visual Studio.
2. Ensure `welcome.wav` is included in the project directory.
3. Run the application.
4. Interact with the chatbot via the console.
